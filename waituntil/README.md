## Installation

### Homebrew

```sh
brew install remino/remino/waituntil
./waituntil
```

### Download

Go to the
[GitHub download page](https://github.com/remino/remutils/releases/latest) for
the latest release, and download the source code from there.

### Git clone

```sh
git clone git@github.com:remino/remutils.git
cd remutils/waituntil
./waituntil
```

## Usage

Run `man waituntil` or `waituntil -h` to see how to use it.
