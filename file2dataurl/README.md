## Installation

### Homebrew

```sh
brew install remino/remino/file2dataurl
./file2dataurl
```

### Download

Go to the
[GitHub download page](https://github.com/remino/remutils/releases/latest) for
the latest release, and download the source code from there.

### Git clone

```sh
git clone git@github.com:remino/remutils.git
cd remutils/file2dataurl
./file2dataurl
```

Run `file2dataurl` without arguments to see how to use it.
